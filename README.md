# 玄精道 —— 唯心传统修仙世界观

> **玄是本原，精是玄的智慧具象。先于宇宙存在的精神体通过自我推演而形成了万千自洽的逻辑结构（大道），每一条大道生成一个宇宙。**

本设定是一套完整的唯心主义修仙世界观，从宇宙论、资源体系、境界体系、战斗逻辑到心魔劫数、点化机缘，以及鬼仙兵仙等存在形态，共十二个章节 + 序言。

## 核心理念

- **万物本原是玄，不是物质**——物质是玄的凝聚态，意识（精）是玄的智慧具象
- **修仙不是积累能量，而是拓展认知**——修炼的本质是"格物致知"
- **战斗不是能量对轰，而是世界观的对撞**——赢的逻辑更底层、更包容
- **成仙不是成为什么，而是记起你本来就一直是仙**
